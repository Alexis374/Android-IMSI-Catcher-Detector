Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => third_party/RootShell/src/main/AndroidManifest.xml
* res/ => third_party/RootShell/src/main/res/
* src/ => third_party/RootShell/src/main/java/
